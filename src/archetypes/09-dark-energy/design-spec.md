# 09 — Dark Energy

**Vibe:** High-intensity gym, CrossFit box, personal training, athletic performance
**Best for:** Gyms, fitness studios, CrossFit boxes, personal training, martial arts

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| bg | #0a0a0a | Main background |
| surface | #141418 | Cards |
| green | #00ff41 | Primary accent, CTAs |
| lime | #7fff00 | Secondary highlight |
| navy | #1a1a2e | Deep accent sections |
| text | #e8e8e8 | Body text |
| muted | #888888 | Secondary text |

## Typography

| Role | Font | Weight |
|------|------|--------|
| Display | Bebas Neue | 400 |
| Headings | Oswald | 500-700 |
| Body | Inter | 400-500 |

## Patterns & Effects

- **Motion blur:** CSS gradient tricks suggesting speed/movement
- **Angular diagonals:** Skewed section dividers, diagonal accents
- **Glow effects:** Green text-shadow and box-shadow on accents
- **Dark gradients:** Black to deep navy fades
- **Bold numbers:** Large stat displays (members, classes, results)

## Section Styling

- **Hero:** Black bg, huge condensed headline with green accent, action imagery
- **Program cards:** Dark cards with green accent borders, icon + description + CTA
- **Trainer cards:** Photo-forward cards with green name accents
- **Buttons:** Green fill, black text, sharp, aggressive styling

## Component Guidelines
- **PHOTO-FIRST.** Fitness is visual. Every program/trainer card must have real action photography. Never use emoji icons for fitness services.
- Aggressive, energetic, motivating — this is not a calm yoga studio
- Green is the electric energy color — use for all CTAs and accents
- Program cards: large action photo, program name + description + CTA
- Typography should feel powerful and athletic
- Statistics and numbers should be prominent
- Photography should show action, sweat, community, intensity — real training photos, not posed stock
