# 04 — Dark Gallery

**Vibe:** Art gallery meets tattoo studio, elegant edgy, minimalist gothic
**Best for:** Tattoo parlors, art studios, galleries, piercing shops, high-end barbers

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| bg | #0d0d0d | Main background |
| surface | #1a1a1a | Cards, elevated sections |
| red | #c41e3a | Primary accent, CTAs |
| blood | #8b0000 | Dark accent, hover states |
| white | #f0f0f0 | Text, borders |
| silver | #c0c0c0 | Secondary text |
| line | #333333 | Subtle borders, dividers |

## Typography

| Role | Font | Weight |
|------|------|--------|
| Display | Cormorant Garamond | 500-600 |
| Headings | Playfair Display | 500-700 |
| Body | Cormorant Garamond | 400 |

## Patterns & Effects

- **Gallery whitespace:** Generous padding, image-forward layouts
- **Line art accents:** Thin decorative SVG lines between sections
- **Ink splatter:** Subtle CSS radial-gradient dots as texture
- **Framed imagery:** Images with thin white borders, gallery-style
- **Serif elegance:** High-contrast serif type against dark backgrounds

## Section Styling

- **Hero:** Dark bg, large serif headline, gallery-style photo grid or single hero image
- **Artist/service cards:** White-bordered dark cards, photo-dominant
- **Gallery:** Masonry or grid layout with thin white borders on images
- **Buttons:** White outline on dark, fills red on hover

## Component Guidelines
- Let imagery dominate — this archetype is visual-first
- Use white sparingly as a border/accent against black
- Red is the only color — use it for CTAs only
- Typography must feel editorial and elevated
