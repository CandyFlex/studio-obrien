## What Jarred must verify manually before any pitch

submit fabricated facts. The following are required before sending a Gap
Read to any target above. Each is a 30-second check, not a database pull:

For each Tier-A Gap Target:
1. Open the target URL in a browser. Take one screenshot of the homepage.
   Note: is it mobile-usable, is there a phone number above the fold, is
   there a contact form, is there an "about" page, when was the copyright
   date last updated?
2. Search the business name on Google. Open Google Business Profile. Note:
   is it claimed, photos count, review count + average star rating, hours
   current, categories correct, "heating contractor" / "air conditioning
   contractor" categories present?
3. Look up the business owner's name + email if public. If not public,
   pick a different outreach channel (phone from GBP, or in-person visit
   since Shelby is local).

For the Tier-C Facebook-only business:
1. Open the Facebook page URL above. Note: does it list a phone number,
   hours, services, recent posts, owner-responded reviews?

These manual checks are not busywork. Section 6's non-invasive principle
forbids scraping business private data, and BUS-INTAKE.md already
encodes this rule. The verification step is the human gate before any
Gap Read ships.
